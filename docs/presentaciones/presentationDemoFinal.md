# Sprint 01 - CasaLink.app

## Introducción

CasaLink.app es una plataforma web para la búsqueda de alquileres, enfocada en generar confianza entre inquilinos y propietarios mediante un sistema de valoraciones basado en experiencias anteriores.

---

## Tecnologías utilizadas

-   **Backend:** Node.js, Express, MySQL
-   **Frontend:** React, Tailwind CSS
-   **Autenticación:** JSON Web Tokens (JWT)
-   **Desarrollo:** Buenas prácticas y modularización con ES6

---

## Objetivos del Sprint 01

1. Definir la arquitectura del proyecto
2. Implementar la base de datos en MySQL
3. Configurar el backend con Express y JWT
4. Crear las primeras vistas en React
5. Establecer la estructura del diseño con Tailwind CSS

---

## Base de datos

Se diseñó un esquema relacional en MySQL que incluye:

-   Usuarios (inquilinos y propietarios)
-   Propiedades
-   Valoraciones
-   Mensajería interna

---

## Backend

-   Configuración de Express con rutas modulares
-   Middleware para autenticación con JWT
-   Controladores para manejo de usuarios y propiedades

---

## Frontend

-   Configuración inicial de React con Vite
-   Implementación de Tailwind CSS para la UI
-   Creación de componentes básicos (Navbar, Cards, Forms)

---

## Resultados y próximos pasos
